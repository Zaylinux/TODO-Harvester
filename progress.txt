## Codebase Patterns
- Python 3.13 is available; use `python` (not `python3`) to run scripts
- Use `python -m mypy backlog_scanner.py --strict` for typechecking (mypy must be installed via pip)
- All path handling uses `pathlib.Path` for cross-platform compatibility
- File reading uses `encoding="utf-8", errors="ignore"` to handle decode errors gracefully
- TODO marker regex: `r"(?P<marker>TODO|FIXME|HACK|XXX):?\s*(?P<text>.*)"` with `re.IGNORECASE`
- Exclude matching: check each part of the relative path against exclude patterns using `fnmatch`
- Module detection: `detect_module(file_path, root)` returns cluster name; uses `_detect_python_module`, `_detect_js_module`, `_detect_java_module` helpers; fallback is `parts[0]` (top-level dir) or `"(root)"`
- `ScanResult.by_module(root)` groups items by module name; requires `root: Path` argument
